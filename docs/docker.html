Docker

Docker system

Show disk usage

docker system df

Prune unused data

docker system prune

Prune all unused data

docker system prune -a

Docker run

Run a container

docker run <image_name>

Run a container with a name

docker run --name <container_name> <image_name>

Run a container with a name and a volume

docker run --name <container_name> -v <host_path>:<container_path> <image_name>

Run a container with a name and a port

docker run --name <container_name> -p <host_port>:<container_port> <image_name>

Docker Images

List Docker Images

docker images

Remove Docker Image

docker rmi <image_id>

Remove All Docker Images

docker rmi $(docker images -q)

Docker Containers

List Docker Containers

docker ps -a

Remove Docker Container

docker rm <container_id>

Remove All Docker Containers

docker rm $(docker ps -a -q)

Stop Docker Container

docker stop <container_id>

Stop All Docker Containers

docker stop $(docker ps -a -q)

Start Docker Container

docker start <container_id>

Docker Volumes

List volumes

docker volume ls

Remove volume

docker volume rm <volume_name>

Remove all volumes

docker volume rm $(docker volume ls -q)

Docker inspect

Inspect a container

docker inspect <container_id>

Inspect a volume

docker inspect <volume_name>

Inspect a network

docker inspect <network_name>

Docker interactive shell

Start interactive shell in a container

docker exec -it <container_id> sh

Start interactive shell in a container with bash - if bash is installed

docker exec -it <container_id> /bin/bash
# or
docker exec -it <container_id> bash

Docker logs

Show all logs and follow

docker logs -f $(docker ps -a -q)

Show logs of a container

docker logs <container_id>

Show logs of a container with timestamps

docker logs -t <container_id>

Show logs of a container with timestamps and follow

docker logs -tf <container_id>

Start docker compose

docker compose up -d

Stop docker compose

docker compose down

Restart docker compose

docker compose restart

Show stats of docker compose

docker compose ps

docker compose ps <service_name>

Update docker compose

docker compose pull

Show all logs of docker compose and follow

Show logs of a service in docker compose

Show logs of a service in docker compose with timestamps

Show logs of a service in docker compose with timestamps and follow

docker compose logs -tf <service_name>

Start interactive shell in a service of docker compose

# or

Dockerfile Build

Build Dockerfile

docker build -t <image_name> .
# e.g.
docker build -t gws/my_image .

Build Dockerfile with a tag

docker build -t <image_name>:<tag> .

Build Dockerfile with a tag and a build argument

docker build -t <image_name>:<tag> --build-arg <key>=<value> .

Build Dockerfile with a tag and a build argument from a file

docker build -t <image_name>:<tag> --build-arg <key>=$(cat <file>) .

Dockerfile

Dockerfile example

# Use an official Python runtime as a parent image
FROM python:3.7-slim

# Set the working directory to /app
# The WORKDIR instruction sets the working directory for any RUN, CMD, 
# ENTRYPOINT, COPY and ADD instructions that follow it in the Dockerfile.
# When the directory is not found, it will be created.
WORKDIR /app

# Copy the current directory contents into the container at /app
COPY . /app

# Install any needed packages specified in requirements.txt
RUN pip install --trusted-host pypi.python.org -r requirements.txt

# Make port 80 available to the world outside this container
EXPOSE 80

# Define environment variable
ENV NAME World

# Run app.py when the container launches
CMD ["python", "app.py"]
